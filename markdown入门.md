# Markdown入门

快速写出格式化的文档。注意这里只讲基础语法，具体的呈现效果因不同渲染器而异，比如有的渲染器在渲染无序列表时前边没有圆点

养成好习惯，每类效果前后空一行，段间空一行。有的渲染器不空行会出现问题

---

## 1 标题

### 语法

```
## 二级标题
### 三级标题
...
```

### 效果

## 二级标题
### 三级标题

一级标题一般是文章题目，出现在最上方

或者一篇文章最高用二级标题，不出现一级标题

---

## 2 无序列表

### 语法

```
- 条目
  - 二级条目(空2格或tab)
- 条目
  - 二级条目 
```

### 效果

- 条目
  - 二级条目
- 条目
  - 二级条目

## 3 有序列表

## 6 代码块

### 语法

```
`行内代码`
```

```
\```语言（在这里注明语言会获得语法高亮效果，与渲染器有关）
只有三个`，没有\，这里加\只是为了转义，不然无法很好的显示，下同
代码块
\```
```

### 效果

这是一条`行内代码`条目

```python
# 我是一段python代码块
def madafaka:
    print("madafaka")
```
